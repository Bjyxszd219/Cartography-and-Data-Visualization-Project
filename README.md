# Mapping & Data Visualization Project

This project was completed as part of my coursework at University College London.  
It explores spatial and economic relationships using **QGIS** and **Python**,  
visualizing multilingual distribution and educational indices through thematic mapping.

## Highlights
- Processed 14,000+ records using Python (Pandas, NumPy)
- Created interactive maps and visual dashboards in QGIS
- Analyzed correlation between GDP per capita and language diversity
- Presented visual storytelling on language vitality and policy impacts
